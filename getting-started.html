Getting Started
The following steps are starting points to configure the Identity Platform

1. Install the SecureAuth Connector (only for cloud deployment)

  • Go to Data Stores > Connectors tab to install the SecureAuth Connector
  • Click Open Installer instructions at the bottom of the page. This page contains all the details you need.
Open Connector Installer page See Documentation

2. Add Data Stores

  • Go to Data Stores tab and click Add a Data Store
  • Select the type of data store to connect
  • Set up your data store configurations
Open Data Stores page See Documentation

3. Configure Global MFA Methods

  • Go to Multi-Factor Methods page
  • Globally enable and configure the MFA methods
  • Later, in a policy, you can choose the MFA methods for end user login workflows
Open MFA Methods page See Documentation

4. Configure Policies

  • After completing global MFA settings, go to the Policies page
  • Review the default login policy; adjust as needed to meet your business requirements
  • Add custom login policies with different authentication rules and MFA methods, as needed
Open Policies page See Documentation

5. Add Applications

  • Go to Application Manager
  • Add an application
  • Configure the connection settings for the application
Open Application Manager page See Documentation
