What's New
Here's what's new in SecureAuth Identity Platform 20.06

FIDO2 Device Support in User Logins
New Feature!

FIDO2 (WebAuthn) is now available to support compatible devices for any of your policy workflows. After users successfully register their FIDO2 devices, the authentication protocol is available as an option during the end user login experience.

Expanded Data Store Support *

Additional data store support is available as part of the New Experience.

Beyond AD and SQL, administrators will now see more data store options including Azure AD, Generic LDAP, NetIQ eDirectory, and Oracle DB.

* Availability depends on the Identity Platform deployment type: cloud or hybrid.

Dynamic IP Blocking
New Feature!

Enhanced security is now available in any login workflow via Dynamic IP Blocking.

Further protect your corporate resources against password spraying and other online password attacks by blocking the associated IP addresses for any amount of time you specify.

SecureAuth Connector Enhancements *

Improvements include display of the Connector GUID and Last Check time in minutes, days, and months.
* Available in the Identity Platform cloud deployment.

New UI Experience

The Identity Platform boasts an all new look and feel providing users an intuitive and easy to use interface (UI) with a cleaner and more modern design.
