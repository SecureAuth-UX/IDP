Version 21.04

What's New

Here's what's new in SecureAuth Identity Platform 21.04

New Product!

SecureAuth Identity Store

SecureAuth Identity Store is a lightweight cloud directory with built-in privacy controls. This new product from SecureAuth is a directory allowing you to manage non-core workforce and user identities. Designed for privacy and security with PII controls, user privileges, and lifecycle management.

New Product!

Login for Linux

New SecureAuth endpoint configuration option available to add multi-factor authentication for logins on Linux operating systems.

New Features!

Complex Policies

More security controls in login workflows for complex use cases. Improvements include conditional rule statements (AND, ELSE, OR) in policy configurations to allow, block, or prompt MFA during end user login.

FIDO2 WebAuthn PIN Support

Updates to the FIDO2 WebAuthn global MFA settings include PIN support.

Internal Application Manager

Use classic realms in the New Experience UI and leverage authentication policies.

SecureAuth Mobile SDK

Use our SecureAuth Mobile SDK to integrate multi-factor authentication capabilities from SecureAuth into your organization's mobile app.

Pre-Login Assessment Service

Create a pre-login questionnaire to determine user risk before allowing login access. For example, ask COVID-19 health questions to determine user risk and allow or block users from onsite access to a work computer.
