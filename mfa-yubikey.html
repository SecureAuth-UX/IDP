YubiKey (non-FIDO2)
Users authenticate using FIDO2 (WebAuthn) bound and roaming devices, such as Windows Hello and compliant YubiKeys.

These settings are for non-FIDO2 YubiKeys. For FIDO2-compliant YubiKeys, go back and enable FIDO2 (WebAuthn).
Configure the YubiKey provisioning settings in the Workflow tab in the Classic Experience.
The YubiKey generates an encrypted password for one-time use. Hackers require physical access of the YubiKey to generate the OTP.
The YubiKey generates a six or eight character, one-time password using OATH HOTP, a strong open authentication standard. The action is event-based.
